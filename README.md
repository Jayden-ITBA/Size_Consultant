# Premium Size Consultant Tool

Một công cụ tư vấn size chuyên nghiệp dành cho các cửa hàng thời trang, được xây dựng bằng HTML, CSS và JavaScript.

## Tính năng
- Tư vấn size dựa trên Chiều cao và Cân nặng.
- Đề xuất các lựa chọn mặc Ôm (Tight) hoặc Rộng (Loose).
- Giao diện Premium Dark Mode với hiệu ứng Glassmorphism.
- Responsive hoàn hảo trên mọi thiết bị.

## Demo
Đã triển khai trên Vercel: [https://size-consultant-premium.vercel.app](https://size-consultant-premium.vercel.app)

## Cài đặt & Sử dụng
Chỉ cần tải toàn bộ thư mục về và mở file `index.html` trong trình duyệt của bạn.

## Công nghệ sử dụng
- HTML5
- CSS3 (Vanilla)
- JavaScript (Vanilla)
- Google Fonts (Outfit)
- Vercel (Deployment)
